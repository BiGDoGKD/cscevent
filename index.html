What is Cloud Skill Challenge?

"Cloud Skills Challenge" will be a self-paced competition mainly based on Cloud Computing. This challenge is open for Undergraduates of University of Colombo, in which each registered participant has to complete modules through the Microsoft Learn Platform. Points would be given for each module each contestant completed throughout the given period. Learn and Earn enough points to stay in the Top of the Leader board to win Exclusive Swags from Microsoft!
